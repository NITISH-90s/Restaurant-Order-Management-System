# 🍽️ Restaurant Order Management System

A modern full-stack Restaurant Order Management System built using Flask, HTML, CSS, JavaScript, Bootstrap, SQLite, and Socket.IO.

This project helps restaurants manage:
- Food menu
- Live customer orders
- GST billing
- Invoice generation
- Admin analytics
- Dashboard monitoring

---

# 🚀 Features

## 🍕 Menu Management
- Display restaurant food items
- Food images with pricing
- Add items to cart
- Quantity handling

---

## 🛒 Order Management
- Live order updates
- Store customer orders
- Real-time order display
- Delete completed orders
- Order history tracking

---

## 💳 Smart GST Billing System
- Automatic GST calculation
- Stylish invoice generation
- Final amount calculation
- Download invoice as PDF
- Customer bill preview

---

## 📊 Admin Analytics
- Total orders tracking
- Revenue calculation
- Most sold item analysis
- Sales summary monitoring

---

## 📈 Dashboard
- Attractive admin dashboard
- Navigation buttons
- Quick analytics access
- Modern UI design

---

## 🔐 Authentication
- Login page
- Admin access support
